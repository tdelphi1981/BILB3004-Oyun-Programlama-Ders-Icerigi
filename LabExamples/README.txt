Bu klasor, her hafta lab foylerinde kullanacaginiz baslangic
Python dosyalarini icerir. Dosyalari indirip uzerinde calisin.

------------------------------------------------------------
  CALISTIRMA
------------------------------------------------------------

Dosyalari calistirmak icin:

    python calisma1.py

PyGame gerektiren dosyalar icin (Lab03 ve Lab04):

    uv init pygame_lab
    cd pygame_lab
    uv add pygame-ce
    uv run python calisma1.py

------------------------------------------------------------
  NASIL KULLANILIR
------------------------------------------------------------

1. Lab foyunuzu acin (ornegin Lab01_Python_Temelleri.pdf)
2. Ilgili klasordeki .py dosyasini acin
3. Once temel kodu calistirin (ciktiyi kontrol edin)
4. "GOREV" isaretli bolgeleri bulun
5. Lab foyundeki talimatlara gore GOREV'leri tamamlayin

------------------------------------------------------------
  GOREV ISARETLERI
------------------------------------------------------------

Her dosyada su isaretleri bulacaksiniz:

    # === GOREV X.Y - Gorev Basligi ===
    # TODO: a) Yapilacak islem
    # TODO: b) Yapilacak islem
    # ============================================

- GOREV numaralari lab foyundeki gorev numaralariyla eslesiyor
- TODO satirlari yapmaniz gereken adimlari belirtir
- Yorum icindeki ornek kodlari referans olarak kullanabilirsiniz
